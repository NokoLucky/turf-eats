# **App Name**: Turf Eats

## Core Features:

- Authentication: Firebase Authentication for users, supporting email/password and phone number login.
- Role Selection: Allows users to select their role (Customer, Driver, or Store Owner) after sign-up.
- Restaurant Listings: Displays available restaurants with name, logo placeholder, rating, and category on the home page.
- Menu Display: Shows menu items with name, description, price, and 'Add to Cart' button on the Restaurant Menu Page.
- Cart and Ordering: Allows customers to view cart items, total price, and place orders on the Cart Page.
- Order Tracking: Enables customers to view order history with live status updates and track driver location on a map using Google Maps integration.
- Restaurant and Driver Management: Enables Restaurant Owners and Drivers to view relevant data.

## Style Guidelines:

- Primary color: Vibrant orange (#FF914D) to evoke feelings of warmth, appetite, and community.
- Background color: Light tan (#F5E8DF), offering a clean, inviting, and natural canvas that allows food imagery and interface elements to stand out without overwhelming the senses.
- Accent color: Muted olive green (#A1C181), creating a balanced, appetizing, natural, and rustic contrast against the orange primary and tan background colors.
- Body and headline font: 'PT Sans', a humanist sans-serif offering a modern, clean, and approachable aesthetic suitable for clear information presentation across both headers and content.
- Use outline icons with a hand-drawn feel to match small-town vibe.
- Subtle transitions on button presses and loading sequences
